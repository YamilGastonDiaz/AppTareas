# 📝 Aplicación de Gestión de Tareas

## Características Clave del Proyecto

* **Formularios en Modales**: Interfaz fluida que permite crear y editar tareas dentro de ventanas modales dinámicas sin recargar la página.
* **Control Completo con EF Core**: Gestión total de datos e interacción con la base de datos utilizando **Entity Framework Core**.
* **Carga de Archivos**: Módulo integrado que permite a los usuarios subir y adjuntar documentos o imágenes directamente a sus tareas.
* **Autenticación Segura**: Sistema de usuarios completo con registro, inicio de sesión y gestión de sesiones mediante **ASP.NET Core Identity**.
* **Alertas Modernas**: Integración de la librería de terceros **SweetAlert2** para mostrar confirmaciones y ventanas emergentes estéticas.

---

## Tecnologías y conceptos utilizados

* **Framework**: ASP.NET Core MVC & Web API
* **Acceso a Datos**: Entity Framework Core
* **Base de Datos**: SQL Server
* **Seguridad**: ASP.NET Core Identity
